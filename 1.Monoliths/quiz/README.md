# Online Quiz Application

Based on Deepseek input:
- https://chat.deepseek.com/a/chat/s/4f81d4ec-e7be-4d23-b886-e0edbbd30bcb

A monolithic Flask-based online quiz application with leaderboard functionality.

## Features

- User registration with anonymous option
- Interactive quiz interface
- Real-time progress tracking
- Leaderboard showing top 3 performers
- Responsive design
- SQLite database
